# Triggers

Triggers are events that start a flow or sub-flow.

There are many types of triggers

## Message received

- Usually starts a flow

## Webhook received

Triggered when a webhook is received

## Scheduled

Triggered at a specific time, one or more times using a start datetime and a cron expression

## Return to main flow

When a sub-flow is done, it might return to the main flow that triggered it and return a value or result from it
